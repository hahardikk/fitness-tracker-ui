# 🏋️ Fitness Tracker UI

A modern and responsive Fitness Tracker frontend built with React. It allows users to register, log in, track fitness activities, view analytics, and generate AI-powered recommendations.

## 🚀 Live Demo

Frontend:
https://fitness-tracker-ui.onrender.com

Backend:
https://fitness-tracker-backend-klk2.onrender.com

API Documentation (Swagger):
https://fitness-tracker-backend-klk2.onrender.com/swagger-ui/index.html

---

## ✨ Features

- User Authentication (JWT)
- Secure Login & Registration
- Dashboard Analytics
- Activity Tracking
- AI Fitness Recommendations
- Protected Routes
- Responsive Design
- Context API for State Management
- Axios Interceptors
- React Router

---

## 🛠 Tech Stack

- React
- React Router DOM
- Context API
- Axios
- Tailwind CSS
- Vite

## ⚙️ Environment Variables

Create a `.env` file in the root directory.

```env
VITE_API_URL=http://localhost:8080/api
```

For production:

```env
VITE_API_URL=https://fitness-tracker-backend-klk2.onrender.com/api
```

## 💻 Installation

Clone the repository

```bash
git clone https://github.com/hahardikk/fitness-tracker-ui.git
```

Move into project

```bash
cd fitness-tracker-ui
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```
